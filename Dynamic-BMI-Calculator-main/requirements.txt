# Requirements & Run Instructions for Dynamic-BMI-Calculator
# -----------------------------------------------
# This file contains both the pip requirements and OS-specific
# instructions to set up a virtual environment and run the app.
#
# Notes:
# - Tested with Python 3.10–3.13. Use a recent Python 3.x.
# - It's recommended to create and use a virtual environment for the project.
#
# Example quick summary:
# 1) Create venv, activate, install requirements
# 2) Run: `python -m streamlit run bmi.py`
#
# -----------------
# PowerShell (Windows)
# -----------------
# From the project directory (run in PowerShell):
#
# cd "C:\Users\Manas\Downloads\Dynamic-BMI-Calculator-main\Dynamic-BMI-Calculator-main"
# # Create venv (uses the active python - you can substitute full path to python.exe)
# python -m venv .venv
# # Activate venv (PowerShell)
# .venv\Scripts\Activate.ps1
# # Upgrade pip and essentials
# python -m pip install --upgrade pip setuptools wheel
# # Install packages from this file
# pip install -r requirements.txt
# # Run Streamlit app
# python -m streamlit run bmi.py
#
# If you prefer to install globally for current user (no venv):
# python -m pip install --user -r requirements.txt
#
# -----------------
# Bash (Ubuntu / Linux)
# -----------------
# From the project directory (run in a bash shell):
#
# cd /path/to/Dynamic-BMI-Calculator-main
# python3 -m venv .venv
# source .venv/bin/activate
# python -m pip install --upgrade pip setuptools wheel
# pip install -r requirements.txt
# python -m streamlit run bmi.py
#
# Ubuntu note: If Python3 or pip is not installed use:
# sudo apt update; sudo apt install -y python3 python3-venv python3-pip
#
# -----------------
# Troubleshooting
# -----------------
# - If `streamlit` is installed but `streamlit run` doesn't open the browser, visit
#   http://localhost:8501 or the URL printed in the terminal.
# - To stop the server: press Ctrl+C in the terminal running Streamlit.
#
